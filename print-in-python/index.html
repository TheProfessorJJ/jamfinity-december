Python print() Function

In python, the print() function is used to tell a computer what to write back to us.

In order to that, we need to tell the computer what to print.

Example

Print “Today is a wonderful day” onto the screen:

				
					print("Today is a wonderful day")
#Output: Today is a wonderful day

Definition

The print() function prints a specified message onto the screen.
