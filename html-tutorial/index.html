The first language I had learned was HTML. It is the heart of a all web pages. HTML is extremely easy to learn and is recommend for people who are eager to get into web developing.

Now, what is HTML? The content you are seeing at the moment on your screen, this content is actually provided by HTML. If you Right-Click on the web page, and choose Inspect, you will see the HTML language used to code the web page.

HTML stands for Hypertext Markup Language:

  • A markup language is a computer language systemized to interpret elements within a document through the use of tags. These tags are snippet of code that we will learn in our tutorial.
  • HyperText is a text that has been wrapped with a link. This link is called a HyperLink.

To develop a web page, learning HTML is the first step, even with a little bit of knowledge of HTML can help you troubleshoot a snippet of code or insert a code/ information in a web page. As you progress, there will other languages, such as CSS & Javascript, you may need to use to make your web page visually compelling. However, our tutorial will solely focus on HTML. As a result, our examples web pages will be in their most basic form Only HTML based page.
